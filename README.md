# pokedex_incie

## Como Rodar o Projeto

### `1- git clone git@github.com:JVictorC/test_practice_inicie.git`
### `2- cd pokedex_incie`
### `3- Abra algum emulador IOS ou ANDROID`
### `3- sh run_app.sh`
